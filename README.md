# FELYA LABS Landing

Static Astro landing site for FELYA LABS.

The production output is generated by Astro into `dist/`. No production runtime server is required.

Initial page load is first-party only: local CSS, JavaScript, images, and fonts. There are no analytics, remote fonts, remote embeds, or third-party form requests before user action. Formspark is contacted only after a waitlist submit; local prototype video media is requested only after the play cover is selected.

| Area | Location |
| --- | --- |
| Pages | `src/pages/` |
| Shared layouts | `src/layouts/` |
| Reusable sections | `src/components/` |
| Site data | `src/data/` |
| Legal source text | `src/content/legal/` |
| Static assets | `public/assets/` |
| Local fonts | `public/fonts/` |
| Browser behavior | `public/assets/js/site.js` |
| Dependency inventory | `dependencies.md` |
| Verification inventory | `tests.md` |

## Local Development

Use one of the following package managers. GitHub Actions uses Bun; the local alternatives produce the same static site.

### Bun

```sh
bun --version
bun install --frozen-lockfile
bun run dev --host 127.0.0.1 --port 4321
bun run build
bun run preview --host 127.0.0.1 --port 4321
```

### npm

```sh
npm ci
npm run dev -- --host 127.0.0.1 --port 4321
npm run build
npm run preview -- --host 127.0.0.1 --port 4321
```

### pnpm

The project pins Bun in `package.json` for GitHub Actions. These flags let pnpm run locally without changing that deployment contract or adding a pnpm lockfile.

```sh
pnpm --pm-on-fail=ignore install --lockfile=false
pnpm_config_verify_deps_before_run=warn pnpm --pm-on-fail=ignore run dev -- --host 127.0.0.1 --port 4321
pnpm_config_verify_deps_before_run=warn pnpm --pm-on-fail=ignore run build
pnpm_config_verify_deps_before_run=warn pnpm --pm-on-fail=ignore run preview -- --host 127.0.0.1 --port 4321
```

### Deno

```sh
deno run -A npm:astro dev --host 127.0.0.1 --port 4321
deno run -A npm:astro build
deno run -A npm:astro preview --host 127.0.0.1 --port 4321
```
